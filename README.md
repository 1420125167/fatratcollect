### 胖鼠采集 最好用的采集小工具
- 支持 爬取文章列表
- 支持 爬取文章详情
- 支持 分页列表爬虫
- 支持 站群采集 站群发布
- 支持 微信
- 支持 图片本地化
- Jquery 语法选择 剔除 标签.
- 定时采集/定时发布 
- 插件完全不脱离 Wordpress
- 让不会写代码的人也才采集

## Requirements
- PHP >= 7.0
- QueryList 最新版
